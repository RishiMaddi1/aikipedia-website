# Aikipedia Website

Aikipedia is an AI-powered Wikipedia platform that dynamically generates and presents knowledge using AI models.

## Technologies
- HTML
- CSS
- JavaScript
- Hugging Face Inference API

## Setup Instructions
1. Clone the repository.
2. Add your API integrations in the JavaScript code.
3. Push any changes to GitHub.
